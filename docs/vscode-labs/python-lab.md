# Python Copilot Agent Lab

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** VS Code / Python

## Overview

Build **Soc Ops** — a Social Bingo game for in-person mixers — while mastering VS Code Copilot Agent Mode in a Python stack. Participants find people who match bingo card questions and try to get 5 in a row. The app is built with a FastAPI (uvicorn) backend and leverages Copilot's most powerful agentic features throughout.

**Concepts covered:**
- Context engineering and custom workspace instructions
- Agentic primitives: background agents, cloud agents, and custom agents
- Design-first frontend development with Copilot
- Test-driven development (TDD) workflows with pytest
- Multi-agent orchestration
