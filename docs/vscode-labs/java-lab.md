# Java Copilot Agent Lab

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** VS Code / Java

## Overview

Build **Soc Ops** — a Social Bingo game for in-person mixers — while mastering VS Code Copilot Agent Mode in a Java/Spring Boot stack. Participants find people who match bingo card questions and try to get 5 in a row. This lab brings the full agentic Copilot experience to the Java ecosystem.

**Concepts covered:**
- Context engineering and custom workspace instructions
- Agentic primitives: background agents, cloud agents, and custom agents
- Design-first frontend development with Copilot
- Test-driven development (TDD) workflows with JUnit/Maven
- Multi-agent orchestration

## Prerequisites

- [Java 21 JDK](https://adoptium.net/) or higher
- [Apache Maven 3.9+](https://maven.apache.org/) (or use the included Maven Wrapper)
- VS Code with GitHub Copilot extension
- GitHub Copilot subscription (Free, Pro, Business, or Enterprise)
