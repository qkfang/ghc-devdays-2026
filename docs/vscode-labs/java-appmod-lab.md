# Java App Modernization Workshop

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** VS Code / Java

## Overview

Modernize a real-world legacy Java application using the **GitHub Copilot app modernization** extension. You'll work with an `asset-manager` project — a web app that stores and processes images using AWS S3, RabbitMQ, and PostgreSQL — and transform it from outdated technology to a modern, cloud-native solution.

**What the modernization process covers:**
- Upgrading from **Java 8 → Java 21**
- Migrating from **Spring Boot 2.x → 3.x**
- Adding **health check endpoints** using custom Copilot skills
- **Containerizing** the application for cloud-native deployment

**Concepts covered:**
- Using the GitHub Copilot app modernization extension to assess applications
- Automated runtime and framework upgrades guided by Copilot
- Exposing health endpoints with AI-assisted custom skills
- Containerization with Docker

## Lab Steps

| Step | Title | Est. Time |
|------|-------|-----------|
| 1 | Assess Your Java Application | ~5 min |
| 2 | Upgrade Runtime & Frameworks (Java 8 → 21, Spring Boot 2.x → 3.x) | ~10 min |
| 3 | Expose Health Endpoints | ~15 min |
| 4 | Containerize Applications | ~5 min |

## Prerequisites

- [JDK 8](https://learn.microsoft.com/en-us/java/openjdk/download#openjdk-8) (for running the initial application locally)
- [Maven 3.6.0+](https://maven.apache.org/install.html)
- [Docker](https://docs.docker.com/desktop/)
- VS Code with the GitHub Copilot app modernization extension
- GitHub Copilot subscription (Free, Pro, Business, or Enterprise)
