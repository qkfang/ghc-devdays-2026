# .NET 10 + Blazor WebAssembly – Copilot Agent Lab

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** VS Code / .NET

## Overview

Build **Soc Ops** — a Social Bingo game for in-person mixers — while mastering VS Code Copilot Agent Mode in the .NET ecosystem. Participants find people who match bingo card questions and try to get 5 in a row. This lab brings the full agentic Copilot experience to .NET 10 and Blazor WebAssembly developers.

**Concepts covered:**
- Context engineering and custom workspace instructions
- Agentic primitives: background agents, cloud agents, and custom agents
- Design-first frontend development with Copilot
- Test-driven development (TDD) workflows with xUnit/dotnet test
- Multi-agent orchestration
