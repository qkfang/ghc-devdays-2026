# TypeScript Copilot Agent Lab

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** VS Code / TypeScript

## Overview

Build **Soc Ops** — a Social Bingo game for in-person mixers — while mastering VS Code Copilot Agent Mode. Participants find people who match bingo card questions and try to get 5 in a row. Along the way, you'll learn how to use GitHub Copilot's most powerful agentic features in a TypeScript/Node.js stack.

**Concepts covered:**
- Context engineering and custom workspace instructions
- Agentic primitives: background agents, cloud agents, and custom agents
- Design-first frontend development with Copilot
- Test-driven development (TDD) workflows
- Multi-agent orchestration
