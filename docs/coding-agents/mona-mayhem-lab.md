# Mona Mayhem Lab

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** Coding Agents

## Overview

Build **Mona Mayhem** — a retro arcade-themed **GitHub Contribution Battle Arena** — while learning how to use GitHub Copilot's most powerful agentic features. The app lets users compare GitHub contribution graphs of two developers in a head-to-head battle format, built with Astro v5.

This lab is all about asynchronous, agent-driven development: you'll delegate work to background and cloud agents while you focus on higher-level design decisions.

**Concepts covered:**
- Context engineering: workspace instructions and background agents
- Plan Mode — break down complex tasks before writing any code
- Agent Mode — build full features with multi-file edits
- Design-first theming with Plan + Agent Mode
- Multi-agent workflows: background and cloud agents
- Bonus open-ended challenges
