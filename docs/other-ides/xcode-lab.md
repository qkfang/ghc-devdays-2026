# Xcode Workshop

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** Xcode / Apple

## Overview

Master GitHub Copilot inside Xcode using a **Landmarks** SwiftUI demo app (based on Apple's official SwiftUI tutorial). This workshop takes you through the full range of Copilot features available in Xcode — from basic code completion to advanced Agent Mode multi-file refactoring and MCP server integration.

**Concepts covered:**
- Understanding code with Copilot Chat
- Code completion with inline suggestions
- Agent Mode for multi-file refactoring
- Plan Agent for breaking down complex tasks
- GitHub MCP Server integration
- Copilot Vision (optional)

## Prerequisites

- Xcode 15 or later
- [GitHub Copilot for Xcode](https://github.com/github/CopilotForXcode/releases) extension installed
- GitHub account signed in to GitHub Copilot in Xcode
- GitHub Copilot subscription (Free, Pro, Business, or Enterprise)
