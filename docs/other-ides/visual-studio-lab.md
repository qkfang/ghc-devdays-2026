# .NET 10 + Blazor + Visual Studio 2026

**Type:** Hands-on Lab | **Time:** ~2 hr | **Level:** Intermediate | **Track:** Visual Studio

## Overview

Build **TinyShop** — a complete e-commerce application — while mastering GitHub Copilot inside Visual Studio 2026. The app consists of a .NET Minimal API backend and a Blazor Server frontend. You'll start with a partially completed codebase and use Copilot to complete missing features and enhance the application throughout the lab.

**Concepts covered:**
- Exploring a codebase with GitHub Copilot Chat
- Code completion with Ghost Text (Next Edit Suggestions / NES)
- Enhancing UI with inline chat
- Referencing files and context in chat
- Using custom instructions
- Implementing features with Copilot Agent Mode
- Copilot Vision for UI mockup-to-code
- Debugging with Copilot
- Commit summary descriptions
- Connecting and using MCP servers
- Plan Mode in Agent
- Reusable prompt files
- Delegating tasks to the cloud (cloud agent)
