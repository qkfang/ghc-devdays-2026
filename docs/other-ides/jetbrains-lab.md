# JetBrains Workshop

**Type:** Hands-on Lab | **Time:** ~1 hr | **Level:** Intermediate | **Track:** JetBrains / Java

## Overview

Master GitHub Copilot inside IntelliJ IDEA (and other JetBrains IDEs) using the classic **Spring PetClinic** application. You'll explore how Copilot accelerates understanding an unfamiliar codebase, learn to write effective custom instructions, and build a real new feature using Agent Mode.

**Concepts covered:**
- Code completion in JetBrains IDEs
- Exploring and understanding an existing codebase with Copilot Chat
- Writing and applying custom Copilot instructions
- Agent Mode feature development

## Lab Steps

The workshop is structured as a guided progression through the Spring PetClinic codebase:
1. **Setup** — Clone Spring PetClinic and configure GitHub Copilot in IntelliJ IDEA
2. **Codebase Exploration** — Use Copilot Chat to understand the application structure
3. **Custom Instructions** — Add reusable `.github/copilot-instructions.md` to guide Copilot's suggestions
4. **Agent Mode Development** — Implement a new feature end-to-end using Copilot Agent Mode

## Prerequisites

- IntelliJ IDEA (Community or Ultimate)
- Java 17+ installed
- Git
- GitHub account with Copilot subscription (Free, Pro, Business, or Enterprise)
