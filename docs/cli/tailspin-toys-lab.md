# Tailspin Toys Workshop

**Type:** Hands-on Lab | **Time:** ~2 hr | **Level:** Intermediate | **Track:** CLI

## Overview

Master GitHub Copilot in the terminal using **Tailspin Toys** — a website for a fictional game crowd-funding company. The project features a Flask/SQLAlchemy backend and an Astro/Svelte frontend. You'll work entirely from the command line, using Copilot CLI to streamline every aspect of the development workflow.

**Concepts covered:**
- Setting up and using GitHub Copilot CLI
- Writing custom instructions to guide Copilot in the terminal
- Connecting MCP servers for richer AI context
- Building and using custom agents
- Defining agent skills for AI-assisted delivery pipelines

## Lab Steps

The workshop guides you through the full Tailspin Toys stack from the terminal:
1. **Setup** — Install GitHub Copilot CLI and configure custom instructions
2. **Exploration** — Use Copilot to understand the Flask backend and Astro/Svelte frontend
3. **MCP Servers** — Connect external tools via MCP for expanded Copilot context
4. **Custom Agents** — Build a custom agent tailored to the Tailspin Toys workflow
5. **Agent Skills** — Define skills to automate delivery tasks with AI assistance

## Prerequisites

- Python 3.x (for the Flask backend)
- Node.js (for the Astro/Svelte frontend)
- GitHub Copilot CLI installed and authenticated
- GitHub Copilot subscription (Free, Pro, Business, or Enterprise)
